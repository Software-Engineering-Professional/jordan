# Jordan - Dev Workshop

A vanilla HTML, CSS, and JavaScript project used as a shared codebase for a Git collaboration exercise.

## Viewing the project

Open `index.html` directly in any browser - no build step or server needed.

## File structure

```
jordan/
├── index.html   - page markup (all four components live here)
├── style.css    - all styles and CSS variables
└── script.js    - all interactivity (pre-built site code + four labelled group sections)
```

## What is in the codebase

The page has some pre-built JavaScript for scroll animations and active nav highlighting. It also contains four components with **no group-specific interactivity yet** - that is your team's job.

| Component            | Element ID             | Visible on |
| -------------------- | ---------------------- | ---------- |
| Desktop navbar       | `desktop-nav`          | ≥ 768 px   |
| Mobile navbar        | `mobile-nav`           | < 768 px   |
| Desktop tab switcher | `desktop-tab-switcher` | ≥ 768 px   |
| Mobile tab switcher  | `mobile-tab-switcher`  | < 768 px   |

## Your workflow

1. Clone this repository
2. Read this README
3. Create your group branch: `git checkout -b group-N` (replace N with your group number)
4. Follow the instructions on your spec sheet
5. Commit and push your work
